Codebar Scaner
Escaner de Código de barras
Un programa para escanear códigos de barras… simple no?
Pues sí, eso es lo que hace el programa, escanear códigos de barras, pero… ¿Cuál es el objetivo de escanear un código de barras? Bueno, empecemos con ¿Qué es un código de barras?
Un código de barras es una imagen que identifica a un producto de manera estandarizada y única en todo el mundo. En palabras más simples, es el documento de identificación de un producto.
Sabiendo que es un código de barras, al poder escanearlos, podemos saber a qué producto pertenece conectando la aplicación a una base de datos y así traer de la misma base de datos TODA la información (cargada) perteneciente al producto escaneado. ¿Interesante no?
Este programa permite traer conectarse a una base de datos en la red y de esta manera tener una aplicación con la capacidad de traer la información de cualquier producto escaneado.
